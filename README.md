# team02_todolist

Todolist 
Spring Boot 기반 Docker 컨테이너와 AWS 클라우드 인프라를 활용한 일정 관리 웹 서비스

기술스택
Language : Java 17
Framework : Spring Boot
Database : MySQL 8.0 (RDS)
Container : Docker, Docker Compose
(CL/CD : GitHub Actions)
(Cloud : AWS)

주요 기능
사용자 회원가입 및 로그인
todo 생성 / 수정 / 삭제 /조회
todo 상태관리 (완료/미완료/진행률)
사용자별 todo 데이터 분리

배포
1.Prerequistites
Java 17
Docker 
Docker Compose
Git
